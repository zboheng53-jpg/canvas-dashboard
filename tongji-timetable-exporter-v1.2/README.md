# 同济课表导出（Chrome / Edge 扩展）

此扩展仅在浏览器本地读取当前的同济教学管理系统课表页面，不会将课表上传到网络。

## 安装

1. 解压本文件夹，或保留其原样。
2. 在 Chrome 打开 `chrome://extensions`；在 Edge 打开 `edge://extensions`。
3. 打开“开发人员模式”。
4. 点击“加载已解压的扩展”，选择本文件夹。

## 使用

1. 登录同济大学教学管理系统并打开“个人课表”。
2. 点击浏览器工具栏的“同济课表导出”图标。
3. 点击“导出 Excel”。

切换学年或学期后，请等待课表区域刷新完成再点击导出；扩展会自动等待最多约 4 秒。

导出结果是真实的 `.xlsx` Excel 文件，含“学生课表”和“已选课程”两个工作表。学生课表会保留原页面课程块跨节次（例如第 1–2 节）的合并显示。
