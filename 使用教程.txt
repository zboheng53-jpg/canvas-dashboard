Canvas Dashboard 使用教程
=========================

这份项目的当前安装、运行、测试与数据安全说明统一维护在 README.md。

本地首次安装
------------

1. 安装 Python 3.10 或更高版本。
2. 在项目目录打开 PowerShell，执行：

   py -m venv .venv
   .\.venv\Scripts\python.exe -m pip install -r requirements.txt pytest

本地启动
--------

开发或前台诊断：

   powershell.exe -NoProfile -ExecutionPolicy Bypass -File .\scripts\dev.ps1

常驻本地服务：

   .\.venv\Scripts\python.exe serve.py

然后在浏览器访问：http://127.0.0.1:5000/

如需双击启动，根目录的“启动.bat”仍可自动创建虚拟环境并启动服务；日常开发优先使用上面的 PowerShell 命令，以确保始终使用项目的 .venv。

数据安全
--------

data\ 目录包含账户、凭据、课程、待办和其他运行数据。不要通过删除 data\ 来“重置”程序，也不要把它提交到 Git。需要恢复或迁移数据时，请先阅读 docs\backup-and-restore.md。
